# Stage 7: Price Intelligence

Stage 7 estimates fair market value and deal quality.

## Purpose

- Estimate fair market price using comparables
- Calculate deal delta (asking vs. estimated)
- Rate deal quality (bargain/fair/overpriced)
- Explain pricing with confidence scores

## Inputs

- Normalized listing attributes
- Stage 4 output (for risk adjustments)
- Comparable listings (retrieved dynamically)
